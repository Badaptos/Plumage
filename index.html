Plumage

Modern. Beautiful. Simple.

Who says simple can't be beautiful?

Plumage was born from the thought that not many people need a full-size (aka bloated) CSS Framework. Plumage is meant to have a beautifully modern look while maintaining a simple workflow. Based from Google's Material Design, Plumage was created to feel alive without the distractions of other designs.

Plumage is for those who love to build websites, but donâ€™t love the hassle of using CSS to make it beautiful.

If you want to start creating right out of the box, Plumage is for you!

Features

cloud

Light Weight

In order to maintain our philosophy, Plumage sticks to only the necessary amount of css needed to get started. Similar to skeleton, but adding just that extra touch to add elagance as you would expect from plumage.

aspect_ratio

Responsive

Responsiveness is no longer optional. Plumage was created to be mobile first and responsive to fit all display sizes and resolutions. All the functionality of responsive design, but with all the convenience you need.

grid_on

Grid System

Plumage contains a twelve column grid system. What seperates us from our competition is that we do not lock you to a 960px container. We give you the option between the standard 960 or a modern 1280
