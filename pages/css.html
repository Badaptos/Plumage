Typography

Type is set with rem units so font sizes and relationship with other elements can be responsively set based on html font size property. Plumage doesn't change html font size, but you can do it if needed for your project. All measurments are baseed on 16 px, for example <h1> with 2.0rem font size means 32px

Main font is Roboto and font for headers is Roboto Slab both fints are served by Google. Base font is set at 1rem which is 16px with 1.6 line height. thing like anchors, strong, emphasis, underline are included.

Use typography helper classes to quickly stylize your text:
all uppercase text with .uppercase
all lowercase text with .lovercase
all Capitalized text with .capitalize

Buttons

With Plumage you can style buttons in a number of different ways. Buttons come in rectangular and round shapes. There are four sizes for each button and block styling for rectangular button just add an appropriet class to style it.
.button-xs or .button-crcl-xs - for x-small button.
.button-sm or .button-crcl-sm - for small button
.button-md or .button-crcl-md - for medium button
.button-lgor .button-crcl-lg - for large button
.button-block - for block button
Add disable to add disabled styling. You can also change button appearence to clear or outline by adding classes of .button-clear and .button-outline. You can add icons inside round buttons by adding class of.material-icons and icon of your choice. Button styles applies to form elements but can also be added with.button class



      
      ----Rectangular buttons----

Forms

Plumage.io framework provides a couple of covered tags for styling forms, such as: Form, Fieldset, Legend, Input Text, Input Number, Text Field, Range, Radio Buttons, Select Button, Checkbox, and Switch.

Fieldset and Legend

Fieldset is a area that works like a <section> tag, separating and wrapping different topics inside your forms. The <Legend> is where you want to print the title of the fieldset. You should keep those tags always close to each other.


    <fieldset title="Get in touch">
    
    <legend>Legend Tag</legend>
    
    <p>This is a empity<b><fieldset></b>. The box-shadow doesn't belong to the style, but it's here to show you the area wrapped by this tag.</p>
    
    </fieldset>
