Get Started

First, download the plumage framework from the button below.

File Structure

On this page you get a quick and super simple guide to help you how to set up your PLumage.io and start building fancy websites. By clicking in the button bellow you are going to download the entire Plumage folder, which is a ziped file. Once you extract it, you will get a structure that should look like that:


    Plumage
    │
    ├── css
    │   │
    │   ├── icons
    │   ├── plumage.css
    │   ├── plumage.css.map
    │   └── plumage.min.css
    │
    ├── js
    │   │
    │   ├── plumage.js
    │   └── plumage.min.js

Setting up your head tag

Once you get everything in your local driver, go ahead and open our index.html, which is a starting file that we provide to make it easier to start to have some fun!

Our index.html has already all links that conect your source code to the styling files and scripts. So all you have to do is include your content and pick up the styling options provided by Plumage framework.

The <head> tag

WARNING: Please make sure you are setting the path of those files properly. If you include or remove your HTML file to a different folder, you need fix paths.


      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="ie=edge">

      <title>Your Page's Name</title>

      <!--Loading your own Favicon (logo not provided) -->
      <link rel="icon" href="../images/logo.png">

      <!--Loading the Reset -->
      <link rel="stylesheet" href="../css/reset.css">

      <link rel="stylesheet" href="../css/jquery-ui.css" type="text/css" />

      <!--Loading jQuery CDN -->
      <script src="https://code.jquery.com/jquery-2.2.4.min.js"integrity="sha256-BbhdlvQf/xTY9gja0Dq3HiwQF8LaCRTXxZKRutelT44="crossorigin="anonymous"></script>

      <!--Loading jQuery UI -->
      <script src="../js/jquery-ui.min.js" type="text/javascript"></script>

      <!--Loading Plumage CSS -->
      <link rel="stylesheet" href="../css/plumage.css">

      <!--Loading a custom CSS style -->
      <link rel="stylesheet" href="../css/main.css">

HTML Template

Want to get up and running quickly? Use the template html file provided in the folder or feel free to copy this code.


      <!DOCTYPE html>
      <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <meta http-equiv="X-UA-Compatible" content="ie=edge">
          <title>Plumage | Modern.Beautiful.Simple | A CSS framework</title>
          <link rel="icon" href="images/logo.png">
          <link rel="stylesheet" href="css/reset.css">
          <link rel="stylesheet" href="css/plumage.css">
          <link rel="stylesheet" href="css/main.css">
        </head>
        <body>
          <header class="navbar">
            <div class="container-lg">
              <img src="images/logo.png" alt="logo" class="navbar-logo">
              <h1 class="navbar-brand"><a href="index.html">Plumage</a></h1>
              <!-- MENU BUTTON MUST GO HERE TO ENSURE FUNCTIONALITY -->
              <div class="navbar-menu-icon"></div>
              <!-- MOBILE NAV MUST GO BEFORE DESKTOP NAV IN ORDER TO FUNCTION PROPERLY -->
              <nav class="nav-mobile-menu">
                <ul>
                  <li><a href="#">Get Started</a></li>
                  <li><a href="#">CSS</a></li>
                  <li><a href="#">Components</a></li>
                  <li><a href="#">Contact Us</a></li>
                </ul>
              </nav>
              <nav class="nav-menu">
                <ul>
                  <li><a href="#">Get Started</a></li>
                  <li><a href="#">CSS</a></li>
                  <li><a href="#">Components</a></li>
                  <li><a href="#">Contact Us</a></li>
                </ul>
              </nav>
            </div>
          </header>
Boom!

That's pretty much it! You are ready to start including your content. But first make sure you understand how Plumage's classes and the key tags covered are configurated, in order to make you get all the powerfull resources from this friendly and nice framework.Have fun!
