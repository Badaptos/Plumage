Get Started

First, download the plumage framework from the button below.

File Structure

When you download the plumage file you will recieve a zip folder. You need to unzip it first to get working. Below is a quick run through of what is included in the framework folder.


    Plumage
    │
    ├── css
    │   │
    │   ├── images
    │   ├── jquery-ui.css
    |   ├── reset.css
    │   ├── plumage.css
    │   └── plumage.min.css
    │
    ├── js
    │   │
    |   |── jquery-ui.min.js
    │   ├── plumage.js
    │   └── plumage.min.js
    |
    ├── template.html

Setting up your head tag

Once you get everything in your local driver, go ahead and open the template, which is a starting file that we provide to make it easier to get coding

Our template has everything that you need. So all you have to do is include your content and pick up the styling options provided by Plumage framework.

The <head> tag

WARNING: Please make sure you are setting the path of those files properly. If you change the path of your html file, you need to fix these paths.


      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="ie=edge">
      <title>Your Page's Name</title>
      <!--Loading the Reset -->
      <link rel="stylesheet" href="css/reset.css">

      <link rel="stylesheet" href="css/jquery-ui.css" type="text/css" />

      <!--Loading jQuery CDN -->
      <script src="https://code.jquery.com/jquery-2.2.4.min.js"integrity="sha256-BbhdlvQf/xTY9gja0Dq3HiwQF8LaCRTXxZKRutelT44="crossorigin="anonymous"></script>

      <!--Loading jQuery UI -->
      <script src="../js/jquery-ui.min.js" type="text/javascript"></script>

      <!--Loading Plumage CSS -->
      <link rel="stylesheet" href="css/plumage.min.css">

      <!--Loading a custom CSS style -->
      <link rel="stylesheet" href="css/main.css">

HTML Template

Want to get up and running quickly? Use the template html file provided in the folder or feel free to copy this code.


      <!DOCTYPE html>
      <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <meta http-equiv="X-UA-Compatible" content="ie=edge">
          <title></title>
          <link rel="stylesheet" href="css/reset.css">
          <link rel="stylesheet" href="css/jquery-ui.css">
          <script src="https://code.jquery.com/jquery-2.2.4.min.js"integrity="sha256-BbhdlvQf/xTY9gja0Dq3HiwQF8LaCRTXxZKRutelT44="crossorigin="anonymous"></script>
          <script src="js/jquery-ui.min.js"></script>
          <link rel="stylesheet" href="css/plumage.min.css">
        </head>
        <body>
          <h1>Hello World!</h1>
        </body>
      </html>

As simple as that!

That's pretty much it! You're now ready to start including your content, but first make sure you understand how Plumage uses its html tags and classes to style your website. Have fun!
