Components

Plumage.io has some cool features coded with jQuery, already styled to help developers and save some time in your coding.

Highlight Dialog Box

Our customized framework is providing 9 spots where you can store different contents to be displayed as a highlight window, which in jQuery is known as Dialog. All that you need to do is start creating as many button tags as you want, in order to show and hide the different contents you have.

Then you need include specific classes to each button tag. The classes will create the conextion between the button and the content that you will show. From 1 to 9, you will be adding classes like:

  • .clickDialog1
  • .clickDialog2
  • .clickDialog3
  • .clickDialog4
  • .clickDialog5
  • .clickDialog6
  • .clickDialog7
  • .clickDialog8
  • .clickDialog9
    •

Then you have to wrap your buttons with a div tag that may wrap everything or each button. By the end of this piece of code, your buttons should look like that:


    <div class="col-lg-4 dialogButtons">
    <button class="clickDialog1">
        Send
    </button>
    <button class="clickDialog2">
        Send
    </button>
    <button class="clickDialog3">
        Send
    </button>
    </div>

Finally, you will include in the bottom of your HTML the piece of code that you want to display inside the dialog window. This content should keep hidden until the user clicks in the button. Now all that you need is create a <div> tag wrapping the content you have, and include on that a ID that will connect with the specific button that you created above. Like in the first situation, here you also can use up to 9 items per page:

  • #boxDialog1
  • #boxDialog2
  • #boxDialog3
  • #boxDialog4
  • #boxDialog5
  • #boxDialog6
  • #boxDialog7
  • #boxDialog8
  • #boxDialog9
    •

Then you should include an title attribute inside each div that you have, where you can write the title that you be displayed on the dialog box. A simple content for these div tags could look like that:


    <div id="boxDialog1" title="Title Number 1">
    <img src="../images/01.jpg"/>
    <p>Lorem ipsum !</p>
    </div>

    <div id="boxDialog2" title="Title Number 2">
    <img src="../images/02.jpg"/>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima dolorum, necessitatibus deserunt quia. Labore, optio reiciendis ipsa quia at neque libero enim ipsam odio voluptate, impedit minus dicta omnis! Distinctio!</p>
    </div>

    <div id="boxDialog3" title="Number 3">
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima dolorum, necessitatibus deserunt quia. Labore, optio reiciendis ipsa quia at neque libero enim ipsam odio voluptate, impedit minus dicta omnis! Distinctio!</p>
    </div>
